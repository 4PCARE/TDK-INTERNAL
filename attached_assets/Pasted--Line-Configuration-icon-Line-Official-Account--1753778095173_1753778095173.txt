เพิ่มหน้า Line Configuration โดยให้เข้าผ่าน icon รูปเฟืองที่อยู่ในการ์ด Line Official Account ซึ่งอยู่ในเมนู Platform Integration ในหน้า Line Configuration จะเป็นการสร้าง template master ของ message ต่าง ๆ ที่ต้องการเตรียมไว้ใช้เพื่อตอบลูกค้าที่ทักผ่าน Line มา จากไฟล์ line_carousal_template.xlsx คือตารางที่ฉันต้องการสร้างเป็น table template เอาไว้เพื่อใช้รวมร่างกันเพื่อสร้างเป็น carousal message เพื่อส่งออกให้กับ Line OA ผ่าน Line Messaging API โดยการ Join กันของแต่ ละ Sheet ฉันต้องการให้สร้างเป็น Scripts เพื่อเตรียมไว้สำหรับส่งข้อความ โดยตัวอย่าง Structure ที่จะต้องรวมร่างกันเป็นดังนี้ 
[ตัวอย่างการรวมเป็น TypeScripts เพื่อส่งข้อความตาม Template ที่สร้าง] 
type Action =
  | {
      type: 'uri';
      label: string;
      uri: string;
    }
  | {
      type: 'postback';
      label: string;
      data: string;
    }
  | {
      type: 'message';
      label: string;
      text: string;
    };

interface CarouselColumn {
  thumbnailImageUrl: string;
  title: string;
  text: string;
  actions: Action[];
}

ดังนั้นสรุปได้ว่า เมื่อเข้ามาที่เมนู Platform Integration ในการ์ดของ Line OA ให้เติมปุ่มที่มี icon setting (รูปเฟือง) เมื่อกดไปแล้วมีให้เลือกว่าจะสร้าง Message Template แบบใดบ้าง โดยเบื้องต้นขอให้เลือกเป็น Carousal กับ Image ได้ เมื่อเลือก Carousal แล้วจะเข้าสู่หน้าการสร้าง Carousal Template โดยให้ User สามารถสร้าง Template ได้ด้วยตนเองผ่าน UI ซึ่งมีสิ่งที่กำหนดได้ดังนี้ thumbnail_image_url title text actions_values action_type action_label description
